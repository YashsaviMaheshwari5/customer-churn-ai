# Customer Churn Prediction using Classification Algorithms

## 📌 Project Overview

This project predicts whether a customer is likely to leave a company (churn) or stay using Machine Learning Classification Algorithms.  
The application uses a Flask backend API and a frontend user interface for prediction.

---

# 🚀 Features

- Predict customer churn instantly
- Machine Learning model using Random Forest Classifier
- Flask REST API integration
- Frontend + Backend architecture
- Easy deployment using Vercel and Railway
